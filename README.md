## 🛠️ How to Run Any Project

### Prerequisites
```bash
# Clone the repository
git clone https://github.com/abyan-lab/Python30-Abyan.git

# Navigate into it
cd Python30-Abyan

# Install all dependencies
pip install -r requirements.txt
```

### Run a specific project
```bash
# Example — Day 1
cd Easy/Day01_ToDo_List
python main.py
```

Each project folder has its own README with 
specific instructions if needed.
